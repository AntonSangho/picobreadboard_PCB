# 요구사항 작성을 위한 질문 
- 수집하고자하는 환경데이터: 예) 온도, 습도  
- 휴대가능여부 
- 물리적인 사이즈 : 예) 휴대폰사이즈 , 카드지갑사이즈 
- 사용자가 손으로 제어하는 방식 : 예) 버튼
- 사용자가 동작을 확인하는 형태 : 예) LED, 부저음, 액정디스플레이
- 데이터를 확인하는 시각적인 결과물의 형태 : 예) 로우데이터, 그래프 
- 생각하는 키트의 외관 형태 
- 베터리로 동작하는 경우, 사용가능해야하는 대략적인 시간 
- 제작 수량
